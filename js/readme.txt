If your plugin requires any javascript, place the script in functions.js

This script will be included in http://www.example.com/js/common.js on all pages. After any changes to this file, you will need to press CTRL-F5 in your browser to flush the browser cache of common.js

The Javascript will be optimised and GZipped before sending to the browser, so you can write code that is well indented and readable without having to worry about squashing to save space.
